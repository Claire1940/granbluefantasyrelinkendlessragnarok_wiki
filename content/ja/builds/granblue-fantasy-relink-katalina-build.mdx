### アレスゲージの溜め方
アレスゲージは前後半の2段階に分かれています [sCRz5ewzt1k]。50%の地点に閾値があるため、一度のコンボフィニッシュで半分を超えて溜めることはできません [sCRz5ewzt1k]。そのため、100%にするには複数の短いコンボを組み合わせる必要があります [sCRz5ewzt1k]。

| コンボ入力 | ゲージ上昇量 | 最適な使用場面 |
|---|---|---|
| **X -> Y** (ショートフィニッシュ) | 低 | 閾値直前での微調整 [sCRz5ewzt1k] |
| **X -> X -> Y** (ミドルフィニッシュ) | 25% | 標準的な牽制 [sCRz5ewzt1k] |
| **X -> X -> X -> Y** (フルフィニッシュ) | 50% (閾値まで) | 非交戦時のメインゲージ回収 [sCRz5ewzt1k] |
| **リンクアタック** | 25%以上 | 戦闘中の即時ゲージ回収 [sCRz5ewzt1k] |
| **アビリティフィニッシュ** | 50% | 隙は大きいが大幅に回収可能 [sCRz5ewzt1k] |

### アレスの維持方法
一度アレスを召喚すると、特定のアクションを繋げない限りすぐに消えてしまいます [sCRz5ewzt1k]。アレスを無期限に維持するには、攻撃アビリティをタイトなシーケンスでループさせる必要があります [sCRz5ewzt1k]。

| アクションの種類 | アレスを維持できるか？ | 備考 |
|---|---|---|
| **攻撃アビリティ** | はい | チェインの起点 [sCRz5ewzt1k] |
| **契約の連撃 (Y連打)** | はい | ダメージソースの核 [sCRz5ewzt1k] |
| **リンクアタック** | はい | コンボの受付時間を延長 [sCRz5ewzt1k] |
| **奥義 (SBA)** | はい | 奥義直後に即座にアビリティを繋ぐ必要あり [sCRz5ewzt1k] |
| **補助アビリティ** | いいえ | 即座に攻撃でキャンセルしない限りアレスが消滅 [sCRz5ewzt1k] |
| **回避 / ノックバック** | いいえ | 即座にアレスが消滅。極力避けること [sCRz5ewzt1k] |

<Callout type="warning" title="アレス中断の警告">

カタリナは、強力な攻撃からパーティ全体への無敵付与まで、多彩なアビリティを持っています [sCRz5ewzt1k]。一般的なエンドゲームのレイドでは、高DPSを維持できる攻撃的なセットが好まれます [sCRz5ewzt1k]。

### 推奨アビリティ構成

| アビリティ名 | 役割 | クールタイム | 推奨理由 |
|---|---|---|---|
| **アズールソード** | 補助 / バフ | 非常に長い | 即座にアレスを召喚。ボスのフェーズ移行などでアレスが消えた後のリカバリーに [sCRz5ewzt1k]。 |
| **エンチャントランズ** | DPS / 接近 | 短い | ダメージ上限が高い。アレス召喚中は強力な多段ヒット攻撃になる [sCRz5ewzt1k]。 |
| **フローズンブレイド** | DPS / 遠距離 | 非常に短い | 発生の早い飛び道具。コンボフィニッシュへ繋げやすい [sCRz5ewzt1k]。 |
| **ウィンターズ・レイン** | DPS / スタン | 中 | スタン値が高く、多段ヒットする物理ダメージ [sCRz5ewzt1k]。 |

### 状況に応じた補助スキル
パーティのサポートが必要な場合は、**ウィンターズ・レイン**を以下のいずれかと入れ替えることができます [sCRz5ewzt1k]：
- **ライトウォール**: アレス召喚中、パーティ全体に無敵を付与 [sCRz5ewzt1k]。厄介なボスギミックを無視するのに最適 [sCRz5ewzt1k]。
- **エメラルドシールド**: パーティに防御バフと怯み無効を付与 [sCRz5ewzt1k]。自身が「怯み無効」ジーンを装備していない場合に有用 [sCRz5ewzt1k]。
- **セイクリッドランズ**: 氷結を付与し、敵の動きを止める [sCRz5ewzt1k]。正確な位置取りが必要 [sCRz5ewzt1k]。

### 理想的な限界突破
カタリナの限界突破を厳選する際は、高いアビリティダメージ上限に到達できるステータスを優先してください [sCRz5ewzt1k]。

- **クリティカル確率UP (+20%)** (クリティカル100%達成に必須) [sCRz5ewzt1k]
- **通常攻撃ダメージ上限UP (+20%)** [sCRz5ewzt1k]
- **アビリティダメージ上限UP (+20%)** (エンチャントランズの強化) [sCRz5ewzt1k]
- **アビリティダメージUP (+20%)** [sCRz5ewzt1k]

  </Step>
  <Step num="3" title="アビリティチェイン">
    契約の連撃の叩きつけモーションの直後、硬直をキャンセルして**フローズンブレイド**や**ウィンターズ・レイン**を発動し、アレスの持続時間を更新します [sCRz5ewzt1k]。
  </Step>
  <Step num="4" title="リンクアタックによる橋渡し">

  </Step>
  <Step num="5" title="奥義(SBA)時の注意点">
    奥義を発動した後、直接「契約の連撃」に繋げることはできません [sCRz5ewzt1k]。奥義の直後に必ず攻撃アビリティを挟むことで、アレスを維持したまま戦闘を継続できます [sCRz5ewzt1k]。
