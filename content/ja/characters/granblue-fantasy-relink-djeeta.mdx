<Section icon={Sparkles} title="ジータのクラスレベル（Adept Arts）の仕組みを理解する">

2026年のメタにおいても、**グランブルーファンタジー リリンクのジータ**はその独自の「クラスレベル」システムにより、最も柔軟なキャラクターの一人であり続けています。特化型のユニットとは異なり、ジータの有効性はクラスレベルに直結しています。コンボフィニッシュを成功させることでクラスレベルが上昇し、装備しているアビリティの威力、効果時間、またはヒット数が大幅に強化されます。

ジータを使いこなすリズムには、強力なアビリティを放つ前に、通常攻撃からフィニッシュへのサイクルを絶えず繰り返し、高いクラスレベルを維持することが含まれます。これにより、即座にユーティリティを得るためにアビリティを早く使うか、最大の効果を得るために高いクラスレベルになるまで待つかという、報酬と戦略的深みのあるゲームプレイが生まれます。

| ステータス項目 | 基本値 (Lv 100) | 説明 |
| :--- | :--- | :--- |
| **基本HP** | 3430 | 前衛ファイターとしての標準的な生存能力。 |
| **基本攻撃力** | 677 | 武器補正前のバランスの取れた攻撃力。 |
| **クリティカル確率** | 5% | デフォルトのクリティカルヒット発生率。 |
| **スタン値** | 80 | 敵のゲージを削るのに効果的な数値。 |

<Callout type="tip" title="プロのヒント：アドレナリンラッシュ">
ジータのサポートアビリティ**アドレナリンラッシュ**は、コンボフィニッシュ直後にタイミングよく発動することで「パワーレイズ」の溜め速度を大幅に向上させます。短い攻撃チャンスでDPSを最大化するために、常にこのウィンドウを狙いましょう。
</Callout>

<Section icon={Target} title="最適なジンの構成と武器の進行">

エンドコンテンツで**グランブルーファンタジー リリンクのジータ**の限界を押し広げるには、彼女の多才な性質を補完するジンに焦点を当てる必要があります。ダメージも重要ですが、ジータはクールタイム短縮や奥義ゲージ（SBA）の上昇量を改善するジンの恩恵を非常に大きく受けます。

2026年の最難関コンテンツを進めるにつれ、武器の選択がプレイスタイルを決定します。「覚醒武器」はHPと攻撃力をバランスよく強化しますが、「究極武器（バハムート武器）」は純粋なダメージ出力において最高峰の基準となります。

| ジンの優先度 | 推奨スキル | メリット |
| :--- | :--- | :--- |
| **最優先** | ダメージ上限 | 厳しいダメージ上限を突破するために必須。 |
| **優先** | 有利属性変換 | すべての攻撃を有利属性として扱い、ダメージを最大化。 |
| **ユーティリティ** | 怒涛 | ヒット時にアビリティのクールタイムを短縮。 |
| **サポート** | 高揚 | 奥義ゲージの上昇量を増やし、フルチェインを早める。 |

<Callout type="info" title="武器スポットライト">
ジータの**究極武器**は、HPが45,000以下の場合に攻撃力+50%とダメージ上限+100%という凄まじいブーストを提供します。これは、リーダーボードでの上位を目指す本格的なジータプレイヤーにとっての最終目標です。
</Callout>

</Section>

<Section icon={Shield} title="高度な戦闘戦略">

ジータをマスターするには、単にボタンを押すだけでは不十分です。戦闘の流れを理解し、サポートアビリティをいかに活用するかが重要です。**コンダクション**は高度なプレイの代表例で、自分の奥義ゲージを味方に分配し、フルチェインをより頻繁に誘発させることができます。

強力なデバフを付与してくるボスに対しては、**ヴェール**や**クリアオール**を組み込むことで、全滅を勝利へと変えることができます。ジータは、チームが苦戦している場合にクエストの途中で純粋なアタッカーから専任の「ピュリファイアー（浄化者）」へとピボットできる唯一のキャラクターです。

| 戦略の焦点 | 推奨アビリティ | チームシナジー |
| :--- | :--- | :--- |
| **フルサポート** | パナケイア、リバイヴ、クリアオール、ファランクス | 新しい高難易度レイドの攻略に最適。 |
| **バランス型DPS** | アーマーブレイク、レイジ、デシメイト、レギンレイヴ | 一般的な周回やタイムアタックに理想的。 |
| **タンク/囮** | かばう、ファランクス、スロウ、アーマーブレイク | ナルメアのような紙耐久のアタッカーを守るために使用。 |

<Callout type="success" title="チェインバーストの最適化">
自分の奥義ゲージが100%で、チームメイトが遅れている場合は**コンダクション**を使用しましょう。ゲージを共有することで強制的に4チェインを発生させ、ボスの拘束時間を延ばし、チームのダメージウィンドウを最大化できます。
</Callout>

</Section>
