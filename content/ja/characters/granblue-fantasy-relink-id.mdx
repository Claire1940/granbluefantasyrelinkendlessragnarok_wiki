<Section icon={Sparkles} title="変身メカニクスのマスター">

**グランブルーファンタジー リリンク イド**を使いこなす上で最も重要なのは、「オルタエ・ゲージ」の管理です。このゲージによって、イドがより強力な形態へ移行できるタイミングが決定されます。現在の2026年メタでは、専用ジーン「**オルタエ・ソウル**」の追加により、これらの形態の運用方法が劇的に変化しました。

<CardGrid cols={3}>
  <Card title="人間形態">
    - **目的**: オルタエ・ゲージの蓄積。
    - **立ち回り**: コンボフィニッシュやアビリティ「レギンレイヴ・レクス」などを用いて素早くゲージを溜める。
    - **役割**: 高火力を叩き出すための準備フェーズ。
  </Card>
  <Card title="竜形態">
    - **目的**: 神威一体に移行するための「カウント（マーク）」の獲得。
    - **立ち回り**: コンボフィニッシュやリンクアタックを決め、カウントを4つ溜める。
    - **バフ効果**: 「怯み無効」効果と被ダメージ20%軽減を獲得。
  </Card>
  <Card title="神威一体">
    - **目的**: 最大DPSの出力。
    - **立ち回り**: ダメージ上限が大幅に引き上げられた強力なフィニッシュ攻撃やアビリティを連発する。
    - **状態**: 全ての弱体効果が無効化され、攻撃力が大幅に上昇。
  </Card>
</CardGrid>

専用ジーン「オルタエ・ソウル」は、竜形態および神威一体時の与ダメージを10%強化します。ゲージの消費速度が200%（3倍）に増加するデメリットがありますが、ゲージが完全に空になるまで形態が強制解除されなくなります。これにより、特にリンクタイム中などにおいて、一度の変身中に神威一体フィニッシュを複数回叩き込むことが可能になります。

| 形態の仕様 | 通常時の効果 | 「オルタエ・ソウル」装備時 |
| :--- | :--- | :--- |
| **竜形態時のダメージ** | 通常の闇属性ダメージ | 与ダメージ+10%（ダメージ上限を無視） |
| **神威一体フィニッシュ** | 1回発動すると強制的に形態終了 | ゲージが空になるまで何度でも発動可能 |
| **怯み無効（Stout Heart）** | 竜形態時のみ付与 | 両方の変身形態で常時発動 |
| **ゲージ管理** | 手動での移行管理が必要 | ゲージが空になるまで変身状態を完全固定 |

<Callout type="warning" title="変身のタイミングに注意">
竜形態のゲージが切れるまでにカウントを4つ溜められなかった場合、神威一体には移行できず、強制的に人間形態へと戻ってしまいます。変身を維持するためにも、リンクアタックは最優先で発動しましょう。
</Callout>

<Section icon={Target} title="アビリティ解説と戦闘立ち回り">

イドのアビリティは、現在の形態（フォーム）に応じて性能が変化します。どのアビリティを温存し、どのタイミングで神威一体時に叩き込むかを理解することが、中堅プレイヤーとトップクラスのDPSアタッカーを分ける決定的な差となります。最近のパッチでは、神威一体時専用のアビリティのほぼ全てにおいて、ダメージ上限が上方修正されました。

| アビリティ名 | タイプ | 主な用途・効果 |
| :--- | :--- | :--- |
| **レギンレイヴ・レクス** | 攻撃 | ゲージを大幅に溜める / 神威一体時は極大ダメージ |
| **ラグナロク・フォーム** | 特殊 | 即座に竜形態へ移行 / ゲージを最大までリチャージ |
| **アルカディア** | 弱体 | 前方広範囲の敵に「スロウ」を付与 |
| **アトネメント** | 強化 | 味方のダメージを肩代わり / カウンター発動 |
| **デスバウンド** | 攻撃 | 多段ヒット攻撃 / 神威一体時は非常に高いダメージ上限 |

**実践向けアドバンスド・テクニック:**
- **モーションキャンセル**: ガードボタンを押し続ける、または回避を連発することで、自動で発生する変身アニメーションを意図的に遅らせることができます。ボスが大きく移動しようとしている時や、フェーズ移行（無敵化）を挟む直前などに非常に有効です。
- **リンクタイムとのシナジー**: リンクタイム中はオルタエ・ゲージが演出的・システム的に減少しません。「オルタエ・ソウル」を装備した状態で神威一体中にリンクタイムに突入すると、リンクタイムが終了するまで神威一体フィニッシュを無限に連発することができます。
- **ゲージの再充填（リフレッシュ）**: 神威一体のゲージが残りわずかになったタイミングで「ラグナロク・フォーム」を使用すると、ゲージを最大まで回復させることができ、最強状態の維持時間を実質的に2倍に伸ばせます。

<Callout type="info" title="アビリティの優先度">
「**ラグナロク・フォーム**」は、神威一体のゲージが消えかける極限のタイミングまで温存しておきましょう。これにより、最も強力な神威一体状態を大幅に引き伸ばし、DPS（時間あたりに与えるダメージ）を最大化できます。
</Callout>
