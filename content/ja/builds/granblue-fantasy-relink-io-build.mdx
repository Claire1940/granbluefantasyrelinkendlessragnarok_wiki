<Section icon={Target} title="基本メカニクスとスターゲイズの最適化">

**グランブルーファンタジー リリンクのイオのビルド**において、その潜在能力を最大限に引き出すためには、基本となるメカニクスを理解することが不可欠です。イオは、独自の「スターゲイズ」システムを中心とした高火力・紙耐久の魔法アタッカーです。素早い近接コンボを繰り出す他のキャラクターとは異なり、イオを操作する際は、チャージ魔法を確実に叩き込むための位置取りとタイミングに集中する必要があります。最大の目標は**魔力の渦**を溜めることであり、これによりスターゲイズのチャージ時間を大幅に短縮し、威力を上昇させることができます。

**動画のハイライト:**
- **魔力の渦の管理**: アビリティとチャージ通常攻撃の両方を使用して、効率よく渦を溜める方法を学びます。
- **高速キャスト（クイックキャスト）**: アビリティ使用後に瞬時に高段階のチャージへと繋げる実演解説。
- **モーションキャンセル**: 火力を落とすことなく、長い硬直時間中に回避を使って安全を確保するテクニック。

イオ使いにとって最も重要なテクニックが**クイックキャスト**です。アビリティや最大までチャージした通常攻撃の直後にスターゲイズを放つことで、チャージ速度が劇的に上昇します。さらに、最大段階まで溜めたスターゲイズの射撃後の硬直モーションをそのまま見届ける必要はありません。ビームが発射された瞬間に回避を行って位置を調整することで、硬直（後隙）をキャンセルし、すぐに次のローテーションへと移行できます。

<Section icon={Swords} title="必須アビリティと装備セットの選択">

**グランブルーファンタジー リリンクのイオのビルド**において、どのアビリティを選択するかは、純粋なDPSを重視するか、それともチームのサポート（ユーティリティ）を重視するかによって異なります。イオは回復やバフも扱えますが、その真価は魔力の渦を素早く生成できる攻撃魔法にあります。

| アビリティ名 | 主な用途 | 必須とされる理由 |
| :--- | :--- | :--- |
| **フラワリーセブン** | 広範囲高火力攻撃 | 7つの隕石を降らせる、イオのスキルの中で最も高いダメージを誇る主力技。 |
| **ライトニング** | 行動妨害（CC） | 敵を麻痺させ、厄介なボスアクションやフェーズをスキップ可能にする。 |
| **アイス** | 素早い渦生成 | クールダウンが非常に短く、クイックキャストを誘発するために使用。 |
| **ファイア** | DPS＆デバフ | 防御力-10%を付与。敵との距離に応じて複数回ヒットする。 |
| **魔力の渦** | 即時セットアップ | 即座に魔力の渦を最大まで獲得し、瞬時に最大段階のスターゲイズを放つ。 |

**ヒールウィンド**や**コンセントレーション**といったアビリティも存在しますが、高難易度コンテンツにおいては最適解とは言えません。2026年のメタでは、ジーン「ポーション所持数」が十分な回復手段を提供してくれるほか、コンセントレーションによる「怯み無効（スーパーアーマー）」効果は、他のジーンやクイックキャストのパッシブアビリティ（キャラクター強化）で補う方が効率的だからです。

<Section icon={Shield} title="ジーンの育成手順とステータスの優先順位">

マニアック（Maniac）やプラウド（Proud）といった高難易度に挑戦するにつれ、**グランブルーファンタジー リリンクのイオのビルド**の焦点は、純粋な攻撃力アップから**ダメージ上限**の管理へとシフトしていきます。本作では、すべての攻撃に与えられるダメージの限界値（上限）が設定されています。攻撃力を上げるジーンをいくら追加してもダメージ数値が増えなくなった場合、すでに上限に達していることを意味します。

| ジーンの種類 | 優先度 | 推奨レベル | イオへの影響 |
| :--- | :--- | :--- | :--- |
| **ダメージ上限** | 必須 | レベル 65 | スターゲイズやフラワリーセブンの最大与ダメージを上昇させる。 |
| **高速チャージ** | 極めて高い | レベル 30 | 最大段階スターゲイズに必要なチャージ時間を劇的に短縮する。 |
| **追撃** | 極めて高い | レベル 45 | ダメージ上限を完全に無視した追加ダメージを発生させる。 |
| **有利属性変換** | 極めて高い | 最大値 | すべての攻撃が敵の弱点属性扱いになる。 |
| **高揚** | 中 | レベル 15+ | ヒット数が多いため、1戦中に3〜4回の奥義（SBA）を発動可能にする。 |

### ゲーム序盤〜中盤の戦略
育成の初期段階では、**クリティカル確率**と**渾身**に焦点を当てましょう。これらは、初期のダメージ上限に到達していない段階において、最もコストパフォーマンスの高いジーンです。クリティカル確率を早期に確保するために、クリティカル確率が上昇する武器シリーズを使用してください。

### エンドゲームにおける最適化
プラウド難易度に到達した後は、**ダメージ上限V**のジーンへと移行する必要があります。最大効果であるレベル65に到達させるには、少なくとも4〜5個のダメージ上限ジーンが必要になります。この段階から、シェロカルテの「よろず屋のシェロ」や遺物からしか入手できない「有利属性変換」や「追撃」といった限定ジーンの収集を始めましょう。

<Callout type="success" title="生存のコツ">
常に**ポーション所持数**のジーンをスロットに入れておきましょう。レベル15になると、オールポーションなどの所持数が大幅に増加するため、イオの回復アビリティが不要になり、その枠を攻撃アビリティに回すことができます。
</Callout>

</Section>

<Section icon={Sparkles} title="エンドゲームの最適化と武器選択">

**グランブルーファンタジー リリンクのイオのビルド**における最終目標は、**究極武器（バハムート武器）**の獲得です。この武器は、最大HPが45,000以下の場合に攻撃力とダメージ上限を大幅に強化する「カタストロフィ」バフを付与してくれます。

<Steps>
  <Step num="1" title="クリティカル武器フェーズ">
    **コロッサスケーン・マグナ**を使用して、クリティカル確率を最大化します。これにより、育成段階でのスターゲイズのダメージが安定します。
  </Step>
  <Step num="2" title="覚醒武器の強化">
    プラウド難易度で苦戦する場合は、**覚醒武器**の強化を検討してください。ただし、素材の消費が激しく、最終的には究極武器に置き換わる点に注意が必要です。
  </Step>
  <Step num="3" title="究極武器の周回">
    プラウド難易度の最終クエストを周回し、イオの究極武器を入手します。入手後は、その真の力を引き出すために、所持している他のすべての武器を最大まで強化する必要があります。
  </Step>
</Steps>

| 武器名 | タイプ | 最適な用途 |
| :--- | :--- | :--- |
| **コラールスタッフ** | HP強化/コレクション | 序盤の生存率向上および武器コレクションボーナス。 |
| **コロッサスケーン・マグナ** | クリティカル確率 | 中盤の安定性とクリティカル確率の確保。 |
| **ガンバンテイン** | 究極武器 | 最終的なエンドゲームにおける最強DPSとダメージ上限の突破。 |

<Callout type="info" title="限界突破の優先順位">
キャラクターの限界突破（リミットボーナス）を厳選する際、最優先すべきは**基本攻撃（通常攻撃）のダメージ上限アップ**です。これはスターゲイズの威力に直接影響します。次点で**アビリティダメージ上限アップ**と**クリティカル確率**を狙いましょう。
</Callout>

</Section>

<Section icon={Map} title="高度な戦闘テクニックとローテーション">

イオを本当に使いこなすためには、基本以上のテクニックを習得する必要があります。上級プレイヤーは、緊迫した戦闘中に少しでもDPSを稼ぎ、ユーティリティ貢献度を高めるために特定の「テクニック」を使用しています。

**1スタック・スターゲイズの応用テクニック**
総合的なDPSを向上させる隠れた方法として、魔力の渦を1つ獲得するたびに、最大チャージ（段階4）まで溜める*前*に、段階1のスターゲイズを挟み込む方法があります。これにより、クイックチャージの速度を最大限に活かすことができます。高い集中力が必要になりますが、長期戦においては総ダメージ量が10〜15%向上します。

**ライトニングによるフェーズスキップ**
本作のボスの多くは、オーバードライブに突入した直後に「真の力解放（ダメージカット状態）」を発動するという一定のパターンを持っています。イオはこれを阻止できます。ボスがオーバードライブに入るまさにその瞬間に**ライトニング**を温存しておき、麻痺させます。麻痺直後に**フラワリーセブン**や最大スターゲイズを叩き込むことで、ボスが真の力解放状態に入る前に「ブレイク」させることが可能です。
