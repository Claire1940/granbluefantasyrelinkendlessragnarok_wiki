<Section icon={Target} title="必須スキルの編成とスタンスの仕組み">

フラウのスキル構成は、攻撃的な召喚とユーティリティバフに分かれています。彼女のユニークなメカニズムは、「ザ・デビル」を召喚して強力な攻撃を繰り出すことです。標準的な**フラウのビルド**では、チームをサポートするために純粋なダメージと敵へのデバフのバランスを取ることが推奨されます。

| スキル名 | タイプ | 効果 | 戦闘での価値 |
| :--- | :--- | :--- | :--- |
| **デッドランド** | 攻撃 | 前方にザ・デビルを召喚して攻撃 | 最も純粋なダメージが高いスキル |
| **オール・フォール・ダウン** | デバフ | 防御力を下げ、バフを消去する火球 | ボス戦において必須 |
| **インドミナス** | バフ | フラウにHP吸収（ドレイン）を付与 | 高い生存能力 |
| **キックフェイス** | パリィ | 予兆のある攻撃に対するカウンター | プレイヤースキル依存の防御手段 |

<Section icon={Map} title="最適なコンボルートと限界突破の優先度">

**フラウのビルド**における核心となるゲームプレイループは、ゲージ管理を中心に回ります。強化スキルを連発するために、できるだけ頻繁に**「ストレングス・ゲイン」**状態に入ることを目指します。

<Steps>
  <Step num="1" title="ゲージの生成">
    通常攻撃コンボから**フリップキック**フィニッシュに繋げます。これにより、勢いを失うことなく自動的にスタンスが切り替わり、もう一方のゲージが溜まり始めます。
  </Step>
  <Step num="2" title="ストレングス・ゲインの発動">
    ブルーとオレンジの両方のゲージが最大になると、ストレングス・ゲインが発動します。次の3回のアビリティ使用が強化され、クールダウンを消費しなくなります。
  </Step>
  <Step num="3" title="強化スキルのサイクル">
    ダメージソースとして**デッドランド**、防御デバフとして**オール・フォール・ダウン**を最優先で使用し、HP回復が必要なら**インドミナス**、スロウが必要なら**クリムゾン・ナイトメア**を使用します。
  </Step>
  <Step num="4" title="リセットと繰り返し">
    3回のフリーキャストが終わったら、特殊攻撃の**スタンスフリップ**を使用して周囲の敵にダメージを与え、即座に次のゲージ回収を開始します。
  </Step>
</Steps>

<Callout type="info" title="限界突破の優先度">
限界突破ポイント（MSP）を消費する際は、**アビリティダメージ上限UP**と**クリティカル確率UP**を優先してください。フラウはDPSの大部分を召喚攻撃に依存しているため、これらの特定攻撃の上限を引き上げることが最も高い投資対効果をもたらします。
</Callout>

| 限界突破項目 | 優先度 | 影響度 |
| :--- | :--- | :--- |
| **アビリティダメージ上限** | ★★★★★ | デッドランドとオール・フォール・ダウンの上限を引き上げる |
| **クリティカル確率** | ★★★★☆ | ダメージ出力の安定性を確保する |
| **通常攻撃ダメージ上限** | ★★★☆☆ | スタンス切り替えコンボのダメージを強化する |
| **奥義ダメージ上限** | ★★☆☆☆ | 有用だが、アビリティダメージに比べると優先度は低い |

</Section>
