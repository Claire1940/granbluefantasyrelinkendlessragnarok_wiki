<Section icon={Swords} title="フラウの最適ビルド優先度">

フラウの序盤ビルドは複雑である必要はありません。まずは武器レベルとマスタリーを整え、その後で実際のクエスト性能を高めるシジルへ移行しましょう。実戦で火力の伸びが鈍くなってきたら、**ダメージ上限** を中心に、より厳しい戦闘で重要になる終盤向けの要素を追加していきます。

<Steps>
  <Step num="1" title="まず武器を安定させる">
    手持ちで最も強い武器を使い、しっかり強化し、フラウが未強化のままクエストに入らないようにしましょう。
  </Step>
  <Step num="2" title="基本の動きを整える">
    基本的な攻撃性能、クールダウン補助、そして1〜2個の快適性シジルを加えて、場に長く立てるようにします。
  </Step>
  <Step num="3" title="本物の火力補助を足す">
    火力の伸びが鈍くなったら、ダメージ上限へ移行し、可能であれば ウォーエレメンタル や 追加ダメージ を重ねます。
  </Step>
  <Step num="4" title="モードに合わせて調整する">
    長いボス戦では、生存力と安定性を優先します。周回では快適性を削って、圧力を維持しましょう。
  </Step>
</Steps>

| ビルド段階 | 優先すべきもの | 避けたいもの |
|---|---|---|
| 序盤 | 武器レベル、基本攻撃力、スキル回転率 | 希少素材を早すぎる段階で高級シジルに使うこと |
| 中盤 | 生存力、クールダウン、安定した火力 | 完全な紙装甲ビルド |
| 終盤 | ダメージ上限、ウォーエレメンタル、追加ダメージ | 戦闘の安定性を無視すること |
| 拡張後 | マスタートレイト、厳選トレイト、高難度向け調整 | 他キャラのビルドをそのまま真似ること |

| シジルの種類 | フラウとの相性 | 最適な用途 |
|---|---|---|
| 攻撃 / HP / 防御 | 良い | 序盤の安定性と快適なクリア |
| クールダウン補助 | 良い | 回転率とスキルループを安定させる |
| ダメージ上限 | 非常に良い | 生火力の伸びが鈍くなった終盤で強い |
| ウォーエレメンタル | 非常に良い | 手強い敵への高効率な火力補助 |
| 追加ダメージ | 非常に良い | すでに安定したビルドで価値を発揮する |

<Callout type="warning" title="強化の優先順位">
武器と編成が安定する前に、高級シジルばかり追いかけないでください。フラウは、急いで終盤装備を組むより、着実な成長から得られる恩恵の方が大きいです。
</Callout>

</Section>

<Section icon={Shield} title="フラウの編成例とパーティ役割">

フラウは、ほかのメンバーが彼女の時間を奪う仕事を肩代わりしてくれるときに最も強くなります。もっとも扱いやすい構成は、耐久の高い前衛、遠隔攻撃役、そしてサポート寄りのキャラクターを1人ずつ入れる形です。これにより、圧力を維持しながら、1回のミスで崩れるリスクを減らせます。

<CardGrid cols={3}>
  <Card title="前衛枠">

  </Card>
  <Card title="遠隔枠">
    ユーエル、またはラカム。ボスが動いたり、戦場が乱れたりしても火力を維持できます。
  </Card>
  <Card title="サポート枠">
    カリオストロ、イオ、またはロゼッタ。回復、バフ、制御によってフラウの安全性が上がります。
  </Card>
</CardGrid>

| 枠 | 推奨役割 | フラウに役立つ理由 |
|---|---|---|
| メイン | フラウ | 圧力を維持し、積極的な操作が報われる |
| 前衛 | 耐久型近接 | 危険を受け止め、攻めの隙を作る |
| 遠隔 | 安全な火力役 | 動きの多いフェーズでも火力を維持できる |
| サポート | 回復またはバフ | 長期戦を安定させる |

| モード | 重視すること | 結果 |
|---|---|---|
| ソロ | AIの安定性と生存力 | クリアがより安定する |
| 協力プレイ | バーストタイミングと総合火力 | ボスのフェーズ管理がしやすい |
| 周回 | 速度と継続火力 | クエストの反復が速くなる |
| 高難度 | 安全性と安定性 | 失敗が減る |

<Section icon={Target} title="フラウの育成ルートとFAQ">

フラウは、最初から本命として育成するほど伸びが良くなります。序盤はシンプルな装備、中盤は安定性の強化、そして終盤で 2026 年の拡張コンテンツに必要な要素へ移行する、という流れです。彼女は一度作って終わりのキャラではなく、着実な強化にしっかり応えてくれます。

| 育成目標 | やること | 得られるもの |
|---|---|---|
| 早期解放 | Endless Ragnarok コンテンツを進めてフラウを解放する | プレイアブルとしての本来の価値が開放される |
| 序盤の周回 | 武器と基本シジルを強化する | クエストが速くなり、戦闘も安全になる |
| 中盤 | スキル回転と役割分担を詰める | 周回ルートが安定する |
| 終盤 | ダメージ上限と高級トレイトを伸ばす | ボス火力と伸びしろが向上する |
| 高難度 | 生存力とモード別調整を追加する | Chaos 対応の安定した周回がしやすくなる |

| よくある失敗 | より良い対応 |
|---|---|
| フラウをサポートキャラとして扱う | まずはアタッカーとして育成する |
| 生存力を無視する | 最適化の前に快適性を1段階加える |
| 高級シジルを早く無理に入れる | 先に武器とマスタリーを育てる |
| バランスの悪い編成で挑む | 遠隔枠とサポート枠を追加する |
