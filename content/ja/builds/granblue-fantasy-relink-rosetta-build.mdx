<Section icon={Map} title="ロゼッタのメカニズムとローズ管理の基本">

ロゼッタは設置型の薔薇に依存しているため、一部で過小評価されがちです。しかし、適切な**ロゼッタの最強ビルド**を施すことで、戦場を支配し、圧倒的な多段ヒットコンボを叩き出す司令塔へと変貌します。彼女の基本メカニズムは、最大3本の薔薇を設置し、それらが自動で攻撃しつつ、現在のローズレベル（1〜4）に応じた様々なバフを提供することです。

**動画のハイライト:**
- ローズレベル（1〜4）の詳細な仕様と、それが攻撃範囲やダメージに与える影響。
- 薔薇を素早く設置するための**回避キャンセル**テクニックの解説。
- コンボフィニッシュによるバフの色（黄、青、桃）の視覚的見分け方。
- アイアン・メイデンとスパイラル・ローズのシナジーによる70ヒット以上の実演。

ロゼッタを使いこなすには、薔薇の配置とレベル上げをマスターする必要があります。薔薇は敵に攻撃がヒットするか、「ブーケ」などの特定のアビリティを使用することでレベルアップします。高レベルの薔薇は攻撃範囲が劇的に広がり、1秒あたりのヒット数も大幅に増加します。

| ローズレベル | 範囲表示 | 間隔ごとのヒット数 | バフの効果量 |
| :--- | :--- | :--- | :--- |
| **レベル 1** | 小さな緑の円 | 1ヒット | 5% |
| **レベル 2** | 中くらいの緑の円 | 2ヒット | 10% |
| **レベル 3** | 大きな緑の円 | 3ヒット | 15% |
| **レベル 4** | 最大の緑の円 | 4ヒット以上 | 20% |

<Callout type="tip" title="プロのコツ：モーションキャンセル">
薔薇の設置ボタン（△/Yまたは右クリック）を押した直後に回避を入力することで、設置モーションを大幅に短縮できます。設置判定を残したまま硬直をキャンセルできるため、素早い位置調整や次の攻撃への移行が可能になります。
</Callout>

<Section icon={Swords} title="最大DPSとユーティリティを両立するおすすめアビリティ">

適切なアビリティ選択は、ロゼッタのビルドを最適化する上で最も重要な要素です。ロゼッタにはいくつかのサポート向けアビリティがありますが、エンドコンテンツのメタは**「アイアン・メイデン」**と**「スパイラル・ローズ」**の強力なシナジーを中心に回っています。

| アビリティ名 | 役割 | 採用理由 |
| :--- | :--- | :--- |
| **スパイラル・ローズ** | メインDPS | ロゼッタ自身とすべての薔薇から茨を伸ばし、数秒間で数十回ものヒットを叩き出す。 |
| **アイアン・メイデン** | バフ/追撃付与 | 自身に「怯み無効」を付与し、ロゼッタのあらゆる攻撃に薔薇による追撃ダメージを追加する。 |
| **タイクーン** | ユーティリティ | すべての薔薇をターゲットの位置に再配置し、最大HPの1.5%分の毒効果を付与する。 |
| **ローズバリア** | 防御 | ローズレベルに応じて最大40%の被ダメージカットを付与。高難易度での生存に極めて重要。 |
| **ロスト・ラブ** | バースト | すべての薔薇を爆発させて大ダメージを与える。爆発後、薔薇は即座に再設置される。 |

ロゼッタの「隠されたテクニック」は、スパイラル・ローズを発動する前にアイアン・メイデンを使用することです。スパイラル・ローズはロゼッタの周囲に無数の攻撃判定を発生させるため、アイアン・メイデンの追撃効果がそのすべてのヒットに対して個別に誘発されます。これにより、特にリンクタイム中にはボスのHPゲージを一瞬で削り取るほどの凄まじい多段ヒットダメージを生み出します。

<Section icon={Target} title="コンボフィニッシュとバフ循環のマスター">

ロゼッタはコンボフィニッシュを繰り出すことで、パーティ全体に3種類の異なるバフを付与できます。バフの効果量は、その時点でフィールドに存在する最も高いローズレベルに依存します。完成された最強ビルドでは、ほぼ常にレベル4の薔薇を維持できるため、最大20%の恩恵を得られます。

<Steps>
  <Step num="1" title="バフの色の確認">
    通常攻撃中にロゼッタの足元に表示される魔方陣の円に注目してください。黄色は「防御」、青色は「リジェネ（自動回復）」、桃色は「攻撃」を示しています。
  </Step>
  <Step num="2" title="防御バフ（通常攻撃1回 ＋ フィニッシュ）">
    通常攻撃を1回振った後にフィニッシュを入力します。ボスからの小ダメージを耐えるために、エンドコンテンツで最も価値のあるバフです。
  </Step>
  <Step num="3" title="リジェネバフ（通常攻撃2回 ＋ フィニッシュ）">
    通常攻撃を2回振った後にフィニッシュを入力します。ボスがターゲット不可になる演出中や、戦闘の合間の立て直しに有効です。
  </Step>
  <Step num="4" title="攻撃バフ（通常攻撃3回 ＋ フィニッシュ）">
    通常攻撃を3回振った後にフィニッシュを入力します。味方の装備が整っておらず、ダメージ上限に届いていない場合にのみ使用します。
  </Step>
</Steps>

| バフの種類 | 色 | レベル4時の効果 | 戦術的な使い分け |
| :--- | :--- | :--- | :--- |
| **防御** | 黄色 | 防御力 +20% | ほとんどのエンドコンテンツレイドにおける基本バフ。 |
| **リジェネ** | 青色 | リジェネ（HP持続回復） | HP維持が必要なギミックフェーズに最適。 |
| **攻撃** | 桃色 | 攻撃力 +20% | 装備が未完成の味方をサポートする際に使用。 |

<Callout type="success" title="バフの維持について">
「ロスト・ラブ」を使用して薔薇を爆発させても、すでに付与されているアクティブなバフは消失しません。再設置された薔薇は自動的に直前のバフ状態を引き継ぐため、爆発後に即座にコンボをやり直す必要はありません。
</Callout>

</Section>

<Section icon={Sparkles} title="エンドコンテンツ向けジーンと限界突破の優先度">

ダメージ上限に到達し、ロゼッタのビルドを最大限に活かすためには、特定のジーン構成が必要です。ロゼッタの薔薇は他のキャラクターほど「追撃」ジーンの恩恵を受けにくいため、純粋な火力ジーンよりもアビリティのクールダウン短縮やユーティリティジーンの優先度が高くなります。

| ジーンカテゴリ | 推奨ジーン | 優先度 |
| :--- | :--- | :--- |
| **キャラクター専用** | **薔薇の開花** | **最優先**（薔薇の攻撃間隔を短縮） |
| **ダメージ上限** | ダメージ上限 V+ (x4) | **必須**（エンドコンテンツの火力上限に達するため） |
| **必須ユーティリティ** | ポーション所持数 / 回避性能 | **高**（生存率と立ち回りの強化に不可欠） |
| **攻撃・火力** | 暴君 / 渾身 / 有利属性変換 | **高**（標準的な火力底上げ用） |
| **クールダウン** | クイックアビリティ / 高揚 | **中**（アイアン・メイデンの稼働率を向上） |

**限界突破（レベル100）のおすすめステータス:**
理想的なビルドを目指す場合は、以下のステータスを狙って厳選しましょう：
1. **アビリティダメージ上限 Up** (20%)
2. **通常攻撃ダメージ上限 Up** (20%)
3. **クリティカル確率 Up** (20%)
4. **攻撃力 Up** または **アビリティダメージ Up**

  <Card title="薔薇の開花">
    薔薇の攻撃間隔を30%短縮します。ロゼッタのDPSを向上させる上で、最も単体寄与度の高い必須ジーンです。
  </Card>
  <Card title="明鏡止水 / 回避性能">
    ジャスト回避成功時の無敵時間を延長し、アビリティのクールダウンを短縮します。ロゼッタの防御的な立ち回りと非常に相性が良いです。
  </Card>
  <Card title="ガラスの刃">
    ダメージ上限を30%引き上げるハイリスク・ハイリターンなジーン。回避を完璧にマスターした上級者のみにおすすめします。
  </Card>
</CardGrid>

<Callout type="info" title="「追撃」ジーンに関する注意点">
多くのキャラクターにとって「追撃」は必須メタですが、ロゼッタの薔薇の攻撃には追撃効果が乗りません。追撃ジーンはロゼッタ自身の直接攻撃にしか適用されないため、1〜2個のスロットに留め、3枠も割くのは避けましょう。
</Callout>
