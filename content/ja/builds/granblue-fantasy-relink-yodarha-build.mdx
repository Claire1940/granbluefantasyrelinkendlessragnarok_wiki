### コンボフィニッシュのループ
コンボフィニッシュを4回連続で成功させると、通常攻撃のコンボルートが短縮されます [1]。これにより、ボタンを1回押すだけでフルコンボを繰り出せるようになり、フィニッシュを高速で連打することが可能になります [1]。ヨダルハはコンボフィニッシュの複数回にわたる斬撃モーション中が無敵状態になるため、このループは高いダメージ出力を維持しながら、比類なき生存能力をもたらします [1]。

### 幕（マーク）
コンボフィニッシュを実行するか、リンクアタックの後にフィニッシュを繋げることで、「幕」を獲得できます（最大3つまで蓄積可能） [1]。これらの幕は、ヨダルハのアクティブアビリティのダメージ、効果時間、またはユーティリティを劇的に強化します [1]。

### 溜め強攻撃とカウンター（パリィ）
ヨダルハの特殊攻撃（強攻撃）ボタンを長押しすると、構え（ブロック）状態になります [1]。敵の攻撃がヒットする直前にボタンを離すことで、正確なパリィが発動します [1]。パリィに成功すると、一時的に**追撃**バフ（元の攻撃の25%分の追加ダメージを与える）が10秒間付与されます [1]。このタイミングをマスターできるかどうかが、平均的なプレイヤーと一流のヨダルハ使いを分ける境界線となります [1]。

| メカニクス | 入力 / アクション | 主なメリット |
| :--- | :--- | :--- |
| **コンボフィニッシュ** | 弱攻撃コンボの最後 | **無敵**フレームと幕を1つ獲得 [1] |
| **無限ループ** | フィニッシュ4回成功 | コンボルートがボタン1回押しに短縮 [1] |
| **リンクアタック派生** | リンクアタック -> フィニッシュ | 瞬時に幕を2つ獲得 [1] |
| **カウンターパリィ** | 閃光に合わせて強攻撃を離す | 10秒間、**追撃**バフを付与 [1] |

<Callout type="tip" title="パリィの最適化">
パリィの受付時間は、ヨダルハの体に表示される白い円状の閃光で示されます [1]。この閃光の瞬間に正確にボタンを離すことで、確定でカウンターが発動します [1]。もしタイミングを逃してしまった場合は、素早く回避を行うことで被ダメージを避けることができます [2]。
</Callout>

<Section icon={Sparkles} title="ヨダルハのおすすめアビリティ構成">

適切なアクティブアビリティを選択することは、ダメージ出力を最大化し、チームに不可欠なユーティリティを提供する上で極めて重要です [1]。最適なエンドゲーム構成は、高いバーストダメージ、クールダウン管理、そしてチーム全体の生存能力に焦点を当てています [1, 2]。

### アクティブアビリティ解説

*   **空融**: ヨダルハの主力ダメージアビリティ [1]。ボタンを押し続けることで高速の連続斬撃を繰り出し、その効果時間は所持している幕の数に応じて延長されます [1]。このアビリティは、複数のヒットボックスを持つ大型ボスに対して非常に強力です [1]。
*   **万事虚通**: ヨダルハのスキル回しの要となるアビリティ [1, 2]。幕が0の状態で使用すると、瞬時に幕が最大（3つ）まで溜まります [1]。すでに幕が3つある状態で使用すると、ヨダルハのすべてのアクティブアビリティのクールダウンが即座にリセットされます [1]。
*   **閃空**: リズミカルなボタン入力が必要な、高ダメージの突進アビリティ [1]。光る白い円に合わせてタイミングよく攻撃ボタンを押すことで、コンボが最大12回まで延長され、凄まじいダメージを与えられます [1]。このモーション中は移動して位置調整を行うことも可能です [1]。
*   **百花繚乱**: ヨダルハ自身に**幻影**バフを付与し、次の3回までの被弾を無効化します [1]。幕が3つの状態で発動すると、このバフがパーティ全体に適用されるため、ボスの強力な大技をやり過ごすための極めて貴重な手段となります [1]。

| アビリティ名 | 役割 | 幕（マーク）の効果 |
| :--- | :--- | :--- |
| **空融** | メインDPS | 最大チャンネル時間と総ダメージが増加 [1] |
| **万事虚通** | 支援 / リセット | すべてのアビリティのクールダウンをリセット（幕3つ時） [1] |
| **閃空** | バーストDPS | 総ヒット数とダメージ上限が増加 [1] |
| **百花繚乱** | パーティ防御 | パーティ全体に**幻影**を付与（幕3つ時） [1] |
| **十文字** | 突進 / 接近 | ダメージが35万から最大70万まで増加 [1] |

<Callout type="info" title="採用を避けるべきアビリティ">
**「お返しだ」**（重複するカウンターアビリティ）や**「三位一体」**（攻撃力バフ）の使用は避けましょう [1]。エンドゲームにおいては、攻撃的なジーンによってヨダルハの素のダメージ上限に容易に到達するため、攻撃力バフは非常に非効率的です [1]。
</Callout>

<Section icon={Target} title="段階的ローテーションガイド">

ダメージを最大化し、アビリティのクールダウンを常に回し続けるためには、幕と「万事虚通」の相互作用をマスターする必要があります [1]。

<Steps>
  <Step num="1" title="幕を溜める">
    戦闘開始直後は、コンボフィニッシュを実行するか、リンクアタックを利用して素早く**幕を3つ**溜めます [1]。
  </Step>
  <Step num="2" title="バーストアビリティの解放">
    **「空融」**と**「閃空」**を発動し、幕を消費しながら凄まじいバーストダメージを与えます [1]。
  </Step>
  <Step num="3" title="パーティバフの適用">
    ボスが強力な全体攻撃を繰り出そうとしている場合は、幕が3つの状態で**「百花繚乱」**を発動し、チームを保護します [1]。
  </Step>
  <Step num="4" title="クールダウンリセットの実行">
    幕が3つある状態で**「万事虚通」**を発動し、すべてのアクティブアビリティのクールダウンを即座にリセットします [1]。
  </Step>
  <Step num="5" title="バーストフェーズの再開">
    リセット後すぐに**「空融」**と**「閃空」**を再キャストし、短い時間枠で2倍のダメージを叩き出します [1]。
  </Step>
</Steps>

<Callout type="success" title="リンクタイムのシナジー">
リンクタイム中は、アビリティのクールダウンが劇的に短縮されます [1]。**「空融」**を連続して何度も発動することが可能になり、天文学的なダメージ数値を叩き出すことができます [1]。
</Callout>

</Section>

<Section icon={Shield} title="おすすめ武器と限界突破">

ゲームの厳しいダメージ上限を突破するためには、正しい装備の選択と最適な限界突破（オーバーマスタリー）の確保が不可欠です [1, 2]。

### 武器の育成手順

1.  **クリティカル確率武器（伏燐）**: 序盤から中盤の攻略において最適な選択肢です [1]。クリティカル確率が大幅に上昇するため、エンドゲーム装備を入手する前の段階でも安定してダメージ上限に到達しやすくなります [1]。
2.  **覚醒武器（一文字）**: バランスの良いステータス配分を好み、まだ難易度「PROUD」を解放していない場合の堅実な代替候補です [1, 2]。
3.  **究極武器（ゾシモス）**: プロト・バハムウトから入手できる、文句なしの最終目標（スロット最優秀）武器です [1, 2]。**「カタストロフィ」**と**「ジーンシミュレータ（ジーン強化）」**のスキルを備えており、ダメージとダメージ上限を劇的に引き上げます [1, 2]。

| 武器タイプ | 名前 | 主なスキル | 主な用途 |
| :--- | :--- | :--- | :--- |
| **クリティカル確率** | 伏燐 | クリティカル確率 | 攻略段階＆安定性重視 [1] |
| **覚醒** | 一文字 | 攻撃力＆HP上昇 | 中盤の移行期 [1, 2] |
| **究極** | ゾシモス | カタストロフィ / ジーン強化 | 究極のエンドゲームDPS [1, 2] |

### おすすめの限界突破（オーバーマスタリー）
よろず屋シェロで限界突破を厳選する際は、以下のステータスを優先度順に狙いましょう：
- **通常攻撃のダメージ上限 Up**（目標：+20%） [1]
- **アビリティのダメージ上限 Up**（目標：+20%） [1]
- **クリティカル確率 Up**（目標：+20%） [1]
- **アビリティダメージ Up** または **攻撃力 Up** [1]

<Callout type="warning" title="HP上限に関する注意">
究極武器（ゾシモス）を使用している場合、**「カタストロフィ」**の効果を維持するために最大HPを**45,000未満**に抑える必要があります [1]。「守護」などのHPが上昇するジーンを装備する際は十分に注意してください [1]。
</Callout>

### ジーン選択の戦略

*   **ダメージ上限**: すべてのエンドゲームビルドにおいて必須です [1, 2]。火力を最大化するには、ダメージ上限をレベル65（通常、究極武器によるブースト込みでレベル15のジーンが4枚必要）にする必要があります [2]。
*   **「幻影の覚醒」（専用ジーン）**: この専用ジーンは、アビリティ使用時に75%の確率で幕を維持できるようにします [1, 2]。これにより、「万事虚通」によるアビリティ再使用時間のリセットが極めて安定します [1]。
*   **ガラスの心臓**: ヨダルハに強く推奨されるジーンです [1]。被弾時に気絶（ピヨり）状態になりますが、ヨダルハは複数のパリィ、無敵フレーム、そして「百花繚乱」による幻影を持っているため、極めて安全に立ち回ることができます [1, 2]。
*   **追撃**: 通常のダメージ上限を突破する追加ダメージを発生させ、総合DPSを最大25%向上させます [1, 2]。
