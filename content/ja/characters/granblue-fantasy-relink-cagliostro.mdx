<Section icon={Sparkles} title="キャラクター概要とコアメカニクス">

カリオストロは、2026年の現環境において最も柔軟なユニットの一人です。錬金術の開祖として、彼女のプレイスタイルは特定の通常攻撃コンボを通じて様々な「殺意の具現」を錬成することにあります。単一の役割に固執する多くのキャラクターとは異なり、適切にビルドされた**カリオストロ**は、装備するジーンやスキルに応じて、専属サポートからメインアタッカーまで自在に立ち回ることができます。

彼女の固有メカニクスである**錬金術の開祖**は、4種類の異なるコンボフィニッシャーを繰り出すことを可能にします。各フィニッシャーには、遠距離攻撃から高いスタン値を誇る回転鋸まで、それぞれ明確な役割があります。さらに、チャージ攻撃である**崩壊（Collapse）**は彼女の主要なダメージ源であり、コンボフィニッシャーやリンクアタックの直後に使用することで、チャージ時間を大幅に短縮できます。

**動画ハイライト:**
- **コンボの汎用性**: 4つのコンボルートの解説と使い分け。
- **スキルのシナジー**: ペイン・トレインとファンタズマゴリアを繋いでバフを維持する方法。
- **エンドゲームセットアップ**: 終末武器と最適なジーンスロットの紹介。
- **実戦戦闘**: 高難易度Proudモードのボスに対する実戦的な立ち回り。

<Section icon={Swords} title="コンボルートと崩壊のマスター">

どのコンボを使うべきかを理解することが、中級者とトッププレイヤーの分かれ道です。各コンボは、位置取り、スタン値の蓄積、スキルのクールダウン短縮など、戦闘状況に応じて異なる目的を果たします。

| コンボタイプ | 入力文字列 | 主な利点 | 最適な状況 |
| :--- | :--- | :--- | :--- |
| **コンボA** | □ > △ | 遠距離スパイク / 高速CDR | 大型ボス / スキル回し |
| **コンボB** | □x2 > △ | 円形範囲攻撃のハサミ | 雑魚処理 / 複数敵 |
| **コンボC** | □x3 > △ | 高ダメージの回転鋸 | 機動力の高いボス / 持続火力 |
| **コンボD** | □x4 > △ | 圧倒的スタン値 / 高ダメージ | 部位破壊 / リンクゲージ稼ぎ |

コンボAは、フィニッシャーごとにスキルのクールダウンを3%短縮する専用ジーン**「開祖の真理」**を発動させるための最も速い手段であるため、特に重要です。巨大な敵に対しては、コンボAのスパイクが複数ヒットし、複雑なコンボよりも高いダメージを叩き出すことがよくあります。一方で、コンボC（回転鋸）は、小さく機動力の高い敵に対して圧力を維持するのに最適です。

<Callout type="warning" title="重要">
コンボDは最も高いダメージとスタン値を持ちますが、モーションが長いです。回避キャンセルに自信がない限り、攻撃パターンの速いボスに対して使用するのは避けましょう。
</Callout>

<Section icon={Shield} title="エンドゲームビルド：武器とジーン">

**カリオストロ**で限界に挑むには、ダメージ上限メカニクスに合わせたギアの最適化が不可欠です。2026年現在、エンドゲームビルドの基準は、終末武器と彼女の錬金サイクルを強化する専用ジーンの組み合わせに集約されています。

<Steps>
  <Step num="1" title="終末武器を装備する">
    プロトバハムートから**マグヌム・オプス**を入手します。この武器は「カタストロフィ」効果により、HPが45,000以下の時にダメージ上限と攻撃力を大幅に引き上げます。
  </Step>
  <Step num="2" title="専用ジーンを装着する">
    **「開祖の真理」**（フィニッシャーでCDR）と**「開祖の戦略」**（崩壊で防御ダウン）を優先します。「真理」はファンタズマゴリアの維持に必須です。
  </Step>
  <Step num="3" title="ダメージ上限に到達する">
    **「ダメージ上限V+」**ジーンを4つ装備します。生存力を高めるために「守護」、「ポーション所持数」、「回避性能」などのサブ特性を探しましょう。
  </Step>
  <Step num="4" title="ダメージ追加">
    2〜3つの**「ダメージ追加」**ジーンを追加し、標準のダメージ上限を超えて全ヒットに20%の固定ダメージを上乗せします。
  </Step>
</Steps>

| ジーン名 | レアリティ | 推奨サブ特性 |
| :--- | :--- | :--- |
| **開祖の真理** | S | スキルアサルト / コンボボーナス |
| **ダメージ上限V+** | S | ポーション所持数 / 守護 |
| **渾身** | S | なし（固有） |
| **暴君** | A | 窮地回避 |
| **クイッククールダウン** | A | 連携攻撃 |

<Callout type="info" title="ビルド情報">
**ファンタズマゴリア**を安定して使用している場合、ジーンだけでクリティカル率を100%にする必要はありません。ベースで80%程度を目指せば、バフによって残り20%が補われ、確定クリティカルとなります。
</Callout>

</Section>

<Section icon={Trophy} title="上級者テクニックと進行チェックリスト">

カリオストロを極めるには、ボタン連打以上の技術が必要です。ボスの行動ウィンドウやフレームデータを理解することが重要です。例えば、コンボCやDの硬直を回避キャンセルすることで、フィニッシャーのボーナスを失わずに次のコンボを素早く開始できます。

**上級戦闘テクニック:**
- **窮地回避とのシナジー**: 窮地回避を使用して、ジャスト回避時に無敵とクールダウン短縮を得ます。これにより、ボスの範囲攻撃の中でも強気な立ち回りが可能になります。
- **リンクアタックのタイミング**: リンクアタックは即座に最大チャージの「崩壊」を撃てる状態にします。リンクゲージが溜まったら、これを利用して長いコンボルートをスキップしましょう。
- **限界突破の優先事項**: **通常攻撃ダメージ上限**と**スキルダメージ上限**を優先します。「崩壊」は通常攻撃としてカウントされるため、この上限を引き上げることは極めて重要です。
