<Section icon={Swords} title="おすすめアビリティとスキル回し">

フェリの火力とサポート性能をバランスよく引き出すためには、適切なアビリティの選択が重要です。2026年現在のメタでは、純粋な防御支援よりも、ペットの稼働率を維持できる攻撃的なアビリティが好まれています。

| アビリティ名 | 主な効果 | 必要ペット数 | 最適な使用タイミング |
| :--- | :--- | :--- | :--- |
| **ヒン・リヒテン** | 攻撃UP/クリティカルUP/追撃/無敵 | 1〜4匹 | 最大火力を出すため、必ず4匹召喚時に使用 |
| **パージ・スピリッツ** | 高いスタン値 ＋ ペット全員召喚 | なし | リンクアタックを誘発させるための主力アビリティ |
| **ウムラウフ** | ベッポが周囲を攻撃 ＋ 怯み無効 | なし | 空中コンボを邪魔されずに維持するための必須スキル |
| **シュトラーフェ** | ペット全員召喚 ＋ 高いスタン値 | なし | オンslaught（フィニッシュ）後に素早くペットを再召喚する用 |
| **ベネディクション** | 再生/防御UP/攻撃UP | 1〜3匹 | 難易度PROUD攻略初期などのサポート用 |
| **ブラウ・ゲスペンスト** | 防御DOWN ＋ 強化効果消去（ディスペル） | なし | 防御力が高いボスや、厄介なバフを使用するボス用 |

**ヒン・リヒテン**は、彼女の火力の要となる最重要アビリティです。ペットが4匹いる状態で発動すると、攻撃力50%UP、クリティカル確率30%UP、さらに追撃効果を付与します。また、発動時の無敵時間によって、激しいボスの攻撃フェーズを安全にやり過ごすこともできます。

<Callout type="warning" title="エンドゲームにおけるバフの上限について">
ゲーム終盤（エンドゲーム）では、「ベネディクション」による攻撃力20%UPバフは容易にダメージ上限に達してしまうため、火力貢献度が低くなります。追撃効果を付与できるアビリティや、スロウなどの強力なデバフを持つアビリティを優先した方が効率的です。
</Callout>

</Section>

<Section icon={Target} title="3000万ダメージを叩き出す空中ループテクニック">

多くのプレイヤーはペットを消費する「オンslaught（強攻撃フィニッシュ）」を多用しがちですが、**フェリの最強ビルド**において最も高いDPS（時間あたりのダメージ）を出せるのは「空中攻撃ループ」です。この方法を使えば、ペットを消費するデメリットを回避しながら、高いダメージ上限を維持し続けることができます。

<Steps>
  <Step num="1" title="戦闘開始時のセットアップ">
    **シュトラーフェ**を発動し、即座に回避（ドッジ）でモーションをキャンセルします。これにより、長い予備動作を踏むことなく一瞬でペット全員を場に展開できます。
  </Step>
  <Step num="2" title="自己強化（バフ）の付与">
    「怯み無効」を得るために**ウムラウフ**を、追撃効果を得るために**ヒン・リヒテン**を発動します。これらも発動直後に回避キャンセルを行い、常に動き回れる状態を維持します。
  </Step>
  <Step num="3" title="打ち上げ攻撃">
    打ち上げ攻撃（ジャンプ ＋ △ボタン）を繰り出し、空中へ移行します。
  </Step>
  <Step num="4" title="空中攻撃（空中ループ）の連打">
    空中攻撃（落下攻撃にならないようタイミングよく連打）を繰り返します。この空中攻撃はオンslaughtと同等のダメージ上限を持ちながら、ペットたちが地上で独立して攻撃し続けるため、総火力が大幅に跳ね上がります。
  </Step>
  <Step num="5" title="リンクアタックによるループ維持">
    リンクアタックのチャンスが発生したら即座に発動し、ペットの召喚時間を瞬時にリフレッシュして再び空中ループへ移行します。
  </Step>
</Steps>

<Callout type="success" title="「怯み無効」とのシナジー">
アビリティ「ウムラウフ」によって「怯み無効（スタウトハート）」を得ることで、フェリは敵の攻撃で怯まなくなります。これにより、ボスが軽微な攻撃で叩き落とそうとしてきても、無視して空中で攻撃を維持し続けることができます。
</Callout>

</Section>

<Section icon={Shield} title="おすすめ武器と限界突破">

選択する武器によって、出すことができる最大ダメージ（上限）が決まります。育成段階ではクリティカル確率が上昇する武器が優秀ですが、最終的には「究極武器」一択となります。

| 武器の段階 | 武器名 | 主なメリット |
| :--- | :--- | :--- |
| **ゲーム序盤** | **ホワイトペタル** | 高いクリティカル確率とクリティカルダメージ |
| **ゲーム中盤** | **エセリアルラッシャー** | ステータスのバランスが良く、早期に最大強化しやすい |
| **エンドゲーム** | **ドラング・スピリッツ** | 専用スキル**「カタストロフィ」**：攻撃力+50% ＆ ダメージ上限+100% |

また、キャラクターの**限界突破（オーバーマスタリー）**では、ゲームのダメージ上限システムを突破するために以下のステータスを最優先で狙いましょう。

一度ダメージ上限に達してしまうと、それ以上「攻撃力」のステータスを伸ばしてもダメージは増えません。エンドゲームのビルド構築においては、生の「攻撃力」よりも「ダメージ上限UP」を常に最優先してください。

<Section icon={Sparkles} title="エンドコンテンツ向けおすすめジーン構成">

ジーンの枠をどのように使うかこそが、**フェリの最強ビルド**を完成させる鍵となります。ダメージ上限の確保と、フェリ専用のユーティリティ性能のバランスを意識しましょう。

| ジーン名 | 重要度 | 効果 |
| :--- | :--- | :--- |
| **ダメージ上限 V+** | 必須 | 最大火力を引き出すため、レベル65（最大値）までの強化が必須 |
| **幽幻の絆（専用ジーン）** | 極めて重要 | ペットの与ダメージが上昇し、敵に攻撃DOWN/防御DOWNデバフを付与 |
| **追撃** | 高い | ダメージ上限を突破して追加ダメージを与える |
| **有利属性変換** | 高い | すべての攻撃が敵の弱点属性扱いになり、火力が大幅に向上 |
| **ガラスの天井（ガラスキャノン）** | 高い | 気絶（ピヨり）しやすくなる代わりに、ダメージ上限が30%も大幅上昇 |
| **渾身** | 中 | 現在のHP割合が高いほど、攻撃力が大きく上昇 |
| **アビリティクイック** | 中 | ヒン・リヒテンやウムラウフのクールダウンを短縮し、回転率を上げる |

  </Card>
</CardGrid>

<Callout type="tip" title="クールダウン短縮の最適化">
**「怒涛」**と**「アビリティクイック」**を組み合わせることで、フェリは手数が非常に多いため、アビリティのクールダウンを爆速で回復させることができます。これにより、ほぼすべてのバースト（攻撃チャンス）の窓に「ヒン・リヒテン」を合わせることが可能になります。
