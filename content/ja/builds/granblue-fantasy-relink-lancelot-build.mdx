<Section icon={Swords} title="グランブルーファンタジー リリンクにおけるランスロットビルドの概要">

ランスロットは、**グランブルーファンタジー リリンク** の中でも非常に洗練された高速近接キャラのひとりです。最適なビルドは、派手な一撃を狙うよりも、絶え間ない圧力を維持する方向に寄せるのが基本です。動き続け、攻撃し続け、ダウンタイムを最小限に抑えるほど強さが際立ちます。

公式のキャラクター一覧とバトルシステムは、[character page](https://relink.granbluefantasy.jp/en/characters/) と [battle manual](https://relink.granbluefantasy.jp/en/manual/detail?s=battles) にまとまっています。本格的なエンドゲーム構成を固める前に、基本仕様を確認するならこの2ページが最適です。

<Section icon={Sparkles} title="最適な武器、シジル、ステータス優先度">

ランスロットが効果的に感じられるようになるのに、複雑なビルドツリーは必要ありません。必要なのは、攻撃力の伸び、ヒットの安定性、そして近接距離を保てるだけの安全性のバランスです。そのため、序盤から細かな特殊ボーナスを考えすぎるより、武器選びとシジル計画のほうが重要になります。

アップグレードシステムの確認には公式の [character building manual](https://relink.granbluefantasy.jp/en/manual/detail?s=character_building) が役立ちますし、プラットフォームやバージョンの文脈を確認するなら [Steam store page](https://store.steampowered.com/app/881020/Granblue_Fantasy_Relink/) も便利です。

**優先度表**

| 優先度 | 選ぶもの | 理由 |
|---|---|---|
| 1 | ダメージ上限 | 高速近接のヒットは、見た目の数字よりも実効火力の伸びが重要 |
| 2 | 会心率と基礎攻撃力 | ストーリー、ポストゲーム、協力プレイまで安定した火力を出しやすい |
| 3 | クールダウン支援 | スキル回転が増えるほど圧力を保ちやすい |
| 4 | 生存補助 | 退く回数を減らして近距離を維持しやすくなる |

**段階別のシジル方針**

| 段階 | 重ねたいもの | 後回しでよいもの |
|---|---|---|
| ストーリーと序盤クエスト | 攻撃力、HP、シンプルな補助 | レアな上位シジル |
| 中盤 | 会心率、クールダウン、ダメージ稼働率の向上 | 完璧な厳選ロール |
| 終盤 | ダメージ上限、強力な攻撃シナジー | 弱い穴埋めステータス |
| エンドゲーム周回 | 上位ダメージ枠、生存保険 | 方向性の定まらないステータスの積み重ね |

**おすすめの構成バランス**

| スロット種別 | 推奨する方向性 | 備考 |
|---|---|---|
| 武器 | 可能な範囲で維持できる最高攻撃力 | こまめに乗り換えるより、強化を続けるほうが良い |
| 攻撃シジル | ダメージ上限、または会心支援 | 長期的な伸びが最も大きい |
| 補助シジル | クールダウン、または操作快適性 | 高速ループを滑らかに保ちやすい |
| 防御シジル | HP、軽減、生存補助 | 難しいボスを学ぶときに特に有効 |

<Callout type="warning" title="この落とし穴に注意">
実際の伸びを無視して、純粋な攻撃力だけを追いかけないでください。火力が上限に当たり始めたら、**ダメージ上限** は小さな攻撃力上昇より価値が高くなることが多いです。
</Callout>

<Section icon={Target} title="ランスロットのロードアウト設定方法">

強いランスロット構成は、表計算のようなものではなく、リズムのある動きとして感じられるべきです。信頼できる初動、対象に張り付く手段、そしてボスにリズムを崩されたときの安全弁。この3つがあると、どの難度のクエストでも扱いやすくなります。

<Steps>
  <Step num="1" title="最も速く、強化を維持できる武器を選ぶ">
    まずは、現実的に継続強化できる中で最良の武器を使いましょう。その後、クエスト報酬が良くなった段階で段階的に強化します。ランスロットは頻繁な乗り換えよりも、安定した成長のほうが恩恵が大きいです。
  </Step>
  <Step num="2" title="最も信頼できるダメージスキルから割り当てる">
    いつも使う枠には、最も安定して強いスキルを置きましょう。高速近接キャラは、強いボタンを使わないままにしておくと価値を失いやすいです。
  </Step>
  <Step num="3" title="快適性か機動力の選択肢を1つ足す">
    位置調整、生存、再参戦を素早く助ける手段を最低1つは残しておきましょう。そうすることで、ボス戦での稼働率を高く保てます。
  </Step>
  <Step num="4" title="実戦クエストでループを試す">
    まずは通常クエストを1回回し、その後で調整します。回避が多すぎる、または攻撃の隙を逃しているなら、少し欲張りを減らして快適性を優先しましょう。
  </Step>
</Steps>

**スキルスロットの参考**

| スロット種別 | 役割 | 最適な使いどころ |
|---|---|---|
| メインダメージスキル | 高稼働、安定したヒット性能 | ボスの隙や連携ウィンドウ |
| 接近手段 | 素早く近接距離へ戻る | 移動するボス、のけぞり後の復帰 |
| 安全スキル | 被弾回避、位置の立て直し | 新しい敵の学習時 |
| フレキシブル枠 | 火力、補助、チーム支援 | 挑むクエストに合わせる |

<Callout type="success" title="シンプルなルール">
そのスキルが、対象に張り付くこと、火力を上げること、生存時間を延ばすことのどれにも役立たないなら、最初の本格的なランスロットビルドではたぶんその枠は不適切です。
</Callout>

</Section>

<Section icon={Shield} title="パーティ支援と周回ルート">

ランスロットは、本人がやりたくない役割を他のメンバーが補ってくれるほど強くなります。安定した前線、遠距離担当、そしてクエスト進行を助ける支援寄りの味方が1人以上いると相性が良いです。特に、より難しいコンテンツを周回し始めるとその重要性が増します。

**最適なパーティ支援パターン**

| パーティ役割 | 相性の良い候補 | ランスロットへの利点 |
|---|---|---|
| 前線の土台 | Vane, Siegfried | 場を押さえ、乱戦を安定させる |
| 遠距離圧力 | Eugen, Rackam | ボスが動いても火力を切らさない |
| 支援ユーティリティ | Cagliostro, Io | 安全性と回復力を追加できる |
| 柔軟枠 | Captain, Charlotta | クエストに必要な穴を埋めやすい |

**周回の重点表**

| コンテンツ種別 | 周回するもの | 重要な理由 |
|---|---|---|
| ストーリークエスト | 基本素材と武器強化 | 序盤の進行を滑らかにする |
| ポストゲームクエスト | より良いシジルと基礎資源 | 本格的なビルド調整の開始点 |
| 難しめのボス周回 | 火力重視の強化 | ランスロットの伸びを押し上げる |
| 協力クエスト | 安定したクリアと追加ドロップ | 素材収集の速度が上がる |
